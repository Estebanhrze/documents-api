## Sobre el proyecto

Esta API fue creada para digitalizar y centralizar la gestión documental, reemplazando el proceso manual de búsqueda en archivadores y carpetas físicas por un sistema 
centralizado en base de datos.

### Problema que resuelve

Actualmente la localización de documentos depende de:
- Búsqueda física en archivadores/carpetas
- Dependencia de la memoria de una persona para saber dónde está cada expediente
- Tiempo elevado para encontrar un documento específico
- Riesgo de pérdida, deterioro o duplicidad de archivos
- Dificultad para saber quién accedió o modificó un documento

### Solución

La API centraliza el almacenamiento y la metadata de los documentos en una base de datos, permitiendo:
- Registrar cada documento con datos clave (tipo, fecha, propietario, categoría, etc.)
- Buscar y filtrar documentos en segundos en vez de minutos/horas
- Mantener trazabilidad (quién subió, modificó o accedió a un documento)
- Reducir el riesgo de pérdida o extravío físico
- Sentar la base para digitalizar procesos que hoy dependen de papel
